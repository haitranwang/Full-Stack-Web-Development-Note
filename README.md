# Full-Stack-Web-Development-Note
Note:
1. Check port is opening:
MAC-OS:
```bash
sudo lsof -i -P -n | grep LISTEN
```
Windows:
```bash
netstat -ano | findstr "LISTENING"
```

2. HTTP Status Code:
- 1xx: Informational
- 2xx: Success
- 3xx: Redirection
- 4xx: Client Error
- 5xx: Server Error
Reference: https://developer.mozilla.org/en-US/docs/Web/HTTP/Status
